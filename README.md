# react-label-index

A simple React component to display labels based on the frequency of their occurence. The higher the frequency of a label, the higher the accentuation in the index.

A storybook with some examples can be found [https://florianbuehler.github.io/react-label-index/](https://florianbuehler.github.io/react-label-index/) and the source code at [https://github.com/florianbuehler/react-label-index](https://github.com/florianbuehler/react-label-index).
